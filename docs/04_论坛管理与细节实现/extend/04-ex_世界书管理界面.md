# 04-ex 世界书管理界面

## 1. 概述

本文档是  [02-1-ex_世界书架构](../02_Agent生成与管理/extend/02-1-ex_世界书架构.md)  的补充，定义世界书的 Web 管理界面。该界面仅供管理员在开发模式下使用，支持可视化编辑、触发键测试和 JSON 导入导出。

## 2. 权限与入口

- 仅在  debug: true  时激活。
- 需先通过“开启开发者操作”警告流程。
- 入口位于设置页面左侧分组导航中的“世界书管理”。

## 3. 条目列表

### 3.1 展示方式

左侧为条目列表，右侧为选中条目的编辑区。列表支持按作用域和 Skill 筛选。

每个条目在列表中显示：标题、作用域、触发方式、优先级、启用/禁用状态、创建该条目的 Skill 名称。

### 3.2 筛选与排序

- 按作用域筛选（global / character / chat）
- 按 Skill 筛选
- 按触发方式筛选
- 按启用状态筛选
- 支持搜索标题和内容

## 4. 条目编辑器

### 4.1 编辑功能

选中条目后，右侧显示其全部字段的可编辑表单：

- 标题
- 作用域
- 内容（Markdown 编辑器）
- 触发方式
- 触发键列表（可逐条添加/删除）
- 逻辑组合规则
- 优先级（拖拽或数字输入）
- 插入位置
- 递归触发开关
- 启用状态
- 过期时间

### 4.2 新增条目

点击“新增条目”，右侧显示空白表单。填写后点击保存，条目写入  world_book_entries  表并立即生效。

### 4.3 删除条目

每条目旁有删除按钮。删除需二次确认。

## 5. 触发键测试工具

### 5.1 功能

提供一个模拟上下文输入框，管理员输入模拟的帖子内容或回复文本后，点击“测试触发”，系统列出当前文本会激活的所有世界书条目。

### 5.2 测试结果展示

- 命中的条目列表
- 每个条目的触发原因（匹配了哪个触发键）
- 按优先级排序后的最终注入顺序
- 总 Token 估算值

## 6. JSON 导入导出

### 6.1 导出

- 支持按作用域或全量导出为 JSON 文件。
- 导出的 JSON 格式与条目结构一致，可直接用于备份或迁移。

### 6.2 导入

- 支持上传 JSON 文件导入条目。
- 导入模式：追加 / 覆盖 / 全量替换。
- 导入前进行格式校验，校验失败给出具体错误信息。
