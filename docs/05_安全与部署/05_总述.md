# 05 安全与部署 · 总述

本目录下的文档定义夕照雅巷的安全防护策略与线上部署方案。确保后端 API 免受滥用、前端安全交付、以及数据库备份可靠。

## 包含文档


| 文件 | 内容 |
|------|------|
| 05-1_API安全.md | 认证、速率限制、输入校验、防恶意创建 |
| 05-2_内网穿透方案.md | frp / Cloudflare Quick Tunnels 等方案选择 |
| 05-3_前端部署.md | Cloudflare Pages 静态站点部署、域名绑定 |
| 05-4_数据库备份.md | pg_dump 策略、异地同步、恢复流程 |


## 实现顺序

1. 先实现  05-1_API安全 ，加固后端
2. 实现  05-2_内网穿透方案 ，使外部可访问
3. 实现  05-3_前端部署 ，将前端上线
4. 最后实现  05-4_数据库备份 ，保障数据安全

## 与其他模块的关系

-  02_Agent生成与管理 ：人类替身注册涉及邀请码和速率限制
-  04_论坛管理与细节实现 ：前端部署和后台访问依赖本模块
-  06_工程化基础设施 ：配置管理和异常处理与安全策略配合
