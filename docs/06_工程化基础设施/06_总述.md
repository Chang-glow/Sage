# 06 工程化基础设施 · 总述

本目录下的文档定义夕照雅巷的工程化基础设施，包括 Skill 与插件架构、配置管理、数据库迁移、任务调度、异常处理、测试策略和日志系统。

## 包含文档


| 文件 | 内容 |
|------|------|
| 06-1_Skill与插件架构.md | Skill 注册与统筹、插件接口、可拓展性设计 |
| 06-2_配置管理.md | .env + config.yaml 分离、Prompt 模板管理 |
| 06-3_数据库迁移.md | Alembic 自动迁移流程 |
| 06-4_任务调度与并发.md | asyncio 调度、Semaphore 并发控制 |
| 06-5_异常与重试.md | 指数退避、状态保护与回滚 |
| 06-6_测试策略.md | 单元测试、集成测试、LLM 回归测试 |
| 06-7_日志与可观测性.md | 结构化日志、简易仪表盘 |


## 实现顺序

1. 先建立 Skill 与插件架构，定义系统的可拓展骨架
2. 搭建配置管理系统
3. 建立数据库迁移流程
4. 实现任务调度和异常处理
5. 配置日志系统
6. 编写测试框架

## 与其他模块的关系

本目录的模块为所有上层业务模块提供基础设施支撑。
