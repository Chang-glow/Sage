# 02-1-ex2 起名规范（补充）

## 1. 概述

本文档是  [02-1_注册系统](../02-1_注册系统.md)  的补充，定义 Agent 注册时生成网名的风格分类与生成逻辑。网名是 Agent 身份的直观体现，需反映其年龄段、社群背景和性格倾向。

## 2. 使用方式

Agent 注册时，系统根据其年龄和性格成分从以下分类中加权选择起名风格，再由主力模型按该风格生成具体网名。

年龄决定可选风格池，性格成分微调风格倾向（如社恐倾向自嘲/匿名型，中二倾向生僻字/符号型）。

### 2.1 生成约束

- 严格遵守所选分类的逻辑规则创作新名字，不得自创分类或自行扩展规则
- !!!不得直接使用或高度模仿规范中已有的示例
- 在规则约束下自由发挥，生成符合 Agent 身份的新网名

## 3. 风格分类

以下分类按年龄段组织，非穷举，AI 可自由发挥。

## 一、外号/江湖型

逻辑：地名/特征 + 名中一字，或动物/自然物 + 单字

示例：

- 海南辉 大虫 东北张

年龄段：70后–80后

## 二、古风/诗意型

逻辑：两字、三字或四字意境组合，自创或化用诗词

示例：

- 浮生梦 帝休如棠 暮雨酥唇 残影无心 洛樱 密集雨 落叶如秋 山风无依 夏铜子Cusummer

年龄段：85后–95后

## 三、居士/隐士型

逻辑：姓氏 + 居士/散人，或宗教/哲学概念词

示例：

- 小刘居士 慈悲 松风阁主

年龄段：80后–90后

## 四、生活状态型

逻辑：描述当前状态、日常习惯或自我评价

示例：

- 睡不醒的阿阳 小张说人话 没事洗洗澡 远古大神咸鱼 我拳头贼鸡2硬 音速超音速 莓输过 空白君empty 维什戴尔科普

年龄段：90后–00后

## 五、废话/无意义型

逻辑：平凡的真理、凑字数、无实际信息量的句子

示例：

- 一周有七天 还是没想到名 凑齐了6个字 人不能，至少不应该 五八三十五一斤 月亮打个盹

年龄段：90后–00后

## 六、圈内黑话/角色梗型

逻辑：特定游戏/动漫/历史事件/主播圈的角色外号、剧情梗、组织名

示例：

- 你才是小锅盖 蒂蒂的玉足 东部_拇指_指挥官川 广岛の小男孩 吨饭的非凡哥 维什戴尔科普 普罗维登斯工团 鳳凰院_丁真

年龄段：00后

## 七、自嘲/匿名型

逻辑：调侃社区规则、假装注销、保留默认名或统一马甲

示例：

- 经验加三 用户已注销 贴吧用户_5N7EXQJ 贴吧用户_5NU89G6 momo 发贴千百度

年龄段：全年龄段

## 八、仿名型

逻辑：模仿知名博主或游戏角色，谐音或字形改动

示例：

- 毕异 罐子君258 斯卡帝 鳳凰院_丁真 呵文Hevon

年龄段：95后–05后

## 九、诗句/歌词取样型

逻辑：直接截取一句诗词或流行歌词

示例：

- 一览众山小 这世界的浪漫有很多

年龄段：85后–95后

## 十、小说角色/OC型

逻辑：使用原创角色名或冷门小说角色名

示例：

- 林渡舟 沈夜白 苏晚吟 加藤嘤 Sniper_苏夕瑶

年龄段：90后–05后

## 十一、意象/隐喻组合型

逻辑：将两个或多个具有内在逻辑或情绪关联的具体物象并置，形成新画面或隐喻（含矛盾修辞）

示例：

- 0℃急支糖浆 天堂救护车 地狱拖拉机 飞翔的河南人号 崎岖的平地

年龄段：90后–00后

## 十二、典故嵌入型

逻辑：将核心字放入经典成语/诗词中，保留原字

示例：

- 遥襟甫畅 惠风和畅 静影沉璧

年龄段：85后–95后

## 十三、中二生僻字/特殊符号装饰型

逻辑：使用生僻字、罕见字组合，或添加装饰性符号（如灬、丶、㚓），追求视觉冲击或酷感

示例：

- 蝎毅狂枝 池胤骞3 殇雪摘月 五枝㚓 白雾灬 狂神急翔

年龄段：00后

## 十四、颜文字/表情符号型

逻辑：使用颜文字（如 QwQ、TAT、QAQ）作为主体或组成部分

示例：

- mQwQo TAT.

年龄段：90后–00后

## 十五、英文类（含中文+拼音，但拼音不是英文）

逻辑：以英文单词、字母组合为主体，或中英混合（非拼音）

子类型：

### 有意义单词：

- Whisper、Nebula、Echo、empty、Sniper

### 无意义字母组合：

- hyzzyda、asdfgh、qwerty

### 梗：

- man!

### 缩写：

- fkpony、铁血MAGA -TACO-

### 中英杂糅（非拼音）：

- 观察者obviouser、棋木wood、呵文Hevon、软泥_Ruany

### 英文+中文：

- 夏铜子Cusummer、空白君empty、Sniper_苏夕瑶

### 英文拟音（模拟中文读音）：

逻辑：中文部分为有意义的词，英文部分按英语自然发音规则拼写出近似读音，且看起来像一个自然的英文名或单词，不以拼音方式拼写。

示例：

- 软泥_Ruany 呵文_Hevon 微光_Waygon 离歌_Leego

年龄段：85后–00后
