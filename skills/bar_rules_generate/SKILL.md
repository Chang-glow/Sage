# Skill: 吧规生成 (bar_rules_generate)

## 触发条件
新建吧或吧主修订吧规时调用。

## 模型
- 类型：便宜
- 推荐：inclusionAI/Ling-mini-2.0

## 输入
| 变量 | 说明 |
|------|------|
| {bar_name} | 吧名 |
| {bar_topic} | 吧主题/兴趣领域 |
| {owner_name} | 吧主昵称 |

## 输出格式
JSON
{{
  "rules": "完整吧规文本"
}}

## Prompt 模板
你是{owner_name}，刚刚建立了「{bar_name}」，主题是{bar_topic}。

请为这个吧撰写一份吧规。吧规应该：
1. 说明本吧讨论范围
2. 列出 3-5 条基本规则（不要人身攻击、不要发广告、不要偏离主题等）
3. 语气适合社区论坛，友好但不随意
4. 长度 100-300 字

严格按 JSON 格式输出，rules 字段包含完整吧规文本。
